## 技术架构分析

### 核心技术栈
- **React 18.3.1**: 使用最新的React版本，支持并发特性
- **TypeScript 5.7.2**: 提供类型安全和更好的开发体验
- **Vite 6.2.0**: 现代化构建工具，提供快速的开发服务器
- **Tailwind CSS 3.4.17**: 实用优先的CSS框架
- **React Router DOM 7.3.0**: 客户端路由管理

### 主要依赖库
- **Framer Motion 12.9.2**: 动画库
- **Lucide React 0.331.0**: 图标库
- **Recharts 2.15.1**: 图表库
- **Sonner 2.0.2**: 通知组件
- **Zod 3.24.2**: 数据验证
- **clsx + tailwind-merge**: CSS类名工具
